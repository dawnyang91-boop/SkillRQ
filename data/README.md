# SkillRQ Data

Raw datasets are read from `/Users/sihan/code/skill-rec/data/raw`.

This project directory stores only derived artifacts:

- `processed/`: canonical JSONL files and splits.
- `indexes/`: dense, lexical, codebook, and hypergraph indexes.
- `cache/`: embeddings, code cards, and other reproducible intermediate files.

Raw files should not be copied here.
